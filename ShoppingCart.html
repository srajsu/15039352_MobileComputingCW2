    How can I tick an item off my list?

    One can simply touch the checkbox next to the item in order to tick off the item from a particular shopping list. Touching the checkbox again will untick the item.

    What happens to the list if I had to reboot my mobile phone while doing my shopping?

    The list will be saved in it's current state and one can continue to use the application once the device is operating.

    Is it possible to print my shopping list?

    Yes, one can export the shopping list and send it via email or print it.
